Listen on your phone (no VPN app)

This listen page is a normal public website. You do not need Tailscale (or any VPN) on your phone to open it. The only technical rule: the stream URL must start with https:// because this page is served over HTTPS — otherwise the browser blocks in-page playback.

0. Repo on the Mac mini

If cd ~/personal-radio fails, clone first (SSH if the repo is private): git clone git@github.com:bskthefirst/personal-radio.git ~/personal-radio — see the private repo file docs/MINI_QUICKSTART.md after you pull on another machine.

1. Give Icecast a public HTTPS URL (Cloudflare)

On the Mac mini (Terminal or SSH from your laptop), with Icecast already on port 8000, either run the helper in the station repo after git pull:

chmod +x mac/start_public_quick_tunnel.sh && ./mac/start_public_quick_tunnel.sh

…or install and run by hand:

brew install cloudflared
cloudflared tunnel --url http://127.0.0.1:8000

The command prints a random https://….trycloudflare.com URL. In the station page’s connection box, set Stream URL to that host with your mount, usually:

https://YOUR-SUBDOMAIN.trycloudflare.com/stream

Tap save to this device, then press play on the audio bar. Any phone on Wi‑Fi or LTE works — no tailnet.

The trycloudflare hostname changes every time you restart the command. For a stable link, use a named Cloudflare Tunnel (still free for typical personal use).

2. “Now playing” is optional

If you see “now playing unavailable”, audio can still work once the HTTPS stream URL is set. Full options (second tunnel for the API, or hosted now_playing.json) are in the private repo file docs/CLOUDFLARE_PUBLIC_STREAM.md after you clone/pull.

3. Where this page is hosted

Deployed copy: https://bskthefirst.github.io/kjph-fm-web/ (static HTML only). Source edits live in the private personal-radio repo under docs/; publish with ./scripts/sync-public-listen.sh.

4. Security note

A public tunnel exposes your stream to the internet. Use a strong Icecast password and treat quick-tunnel URLs like temporary passwords.
